Jumping Safari - Scratch Game Concept

Game Overview

A jumping game where the player controls a safari animal (like a kangaroo or gazelle) that must jump over obstacles while answering animal classification questions.

Sprites

Player Sprite

Main character with jumping animation

Obstacles

Standard obstacles to jump over

Animal Quiz Sprites

Appear every 100 points to ask questions

Background

Scrolling safari background that moves as the player progresses

Game Mechanics

1. Main Game Loop

when green flag clicked
set [score] to [0]
switch backdrop to [Safari Background]
play sound [Game Start] until done
forever
move obstacles left
if then
play sound [Crash]
switch backdrop to [Game Over]
play sound [Game Over] until done
stop [all]

2. Jumping Mechanism

when [space] key pressed
if > then
play sound [Jump]
change y by (20)
glide (0.5) secs to x: (x position) y: (0)

3. Scoring System

when green flag clicked
forever
if <[Obstacle] reaches left edge> then
change [score] by (10)
if <(score) mod (100) = 0> then
create clone of [Animal Quiz Sprite]

4. Quiz Animal Mechanics

when I start as a clone
go to x: (240) y: (pick random -100 to 100)
show
repeat until
move (-5) steps
if then
ask [Is this animal a carnivore, herbivore, or omnivore?] and wait
if > then
change [score] by (-50)
play sound [Wrong Answer]
play sound [Correct Answer]
delete this clone

Variables

score

Tracks player's current score

highscore

Stores the highest score achieved

speed

Controls obstacle movement speed

Sounds

  • Jump: Boing sound effect
  • Crash: Collision sound
  • Game Start: Safari-themed music
  • Game Over: Sad trombone
  • Correct Answer: Cheerful chime
  • Wrong Answer: Buzzer sound

Game Flow

  1. Player clicks green flag to start
  2. Obstacles start moving from right to left
  3. Player presses spacebar to jump over obstacles
  4. Each successful obstacle pass adds 10 points
  5. Every 100 points, a quiz animal appears
  6. If player touches quiz animal, they must answer a question
  7. Wrong answers deduct 50 points
  8. Game ends when player hits an obstacle
  9. High score is displayed on game over screen

Sample Quiz Questions

Animal: Lion

Question: Is a lion a carnivore, herbivore, or omnivore?

Correct Answer: carnivore

Animal: Elephant

Question: Is an elephant a mammal, amphibian, or reptile?

Correct Answer: mammal

Animal: Frog
